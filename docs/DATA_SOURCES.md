# Data Sources

This strategy uses a mix of local data extraction from installed trading software and online API fallbacks. Local extraction is preferred for data quality and reliability.

## Local Extraction (Primary)

### 活跃市值 0AMV — 指南针 Compass

File: `D:\Program Files (x86)\zhinanzhen\ANALYSE\Data\ChinaStk\Z_SK\day.vdat`

Format: Fixed 20-byte records — date (Excel serial, int32 LE), open/close/high/low (float32 LE).

Steps:
1. Close 指南针 software to release file locks
2. Parse `day.vdat` into OHLC + pct_change
3. Save as `data/amv_daily.csv`

Reference: `scripts/extract_amv_klines.py`

### 概念资金流向 — 通达信 TDX

Concept definitions: `D:\new_tdx\T0002\hq_cache\infoharbor_block.dat`
- Text file with `#GN_` headers, GBK encoding
- Contains 269+ concept definitions with constituent stock lists

Concept flow data is generated by combining concept definitions with market trend estimates.
Output: `data/concept_flow.csv`

### ETF 日线行情 — 通达信 TDX（替代 AKShare）

Files: `D:\new_tdx\vipdoc\sh|sz\lday\*.day`

- 支持沪深两市所有 ETF（上交所 `sh51/58*.day`，深交所 `sz159*.day`）
- 32 字节/条定长记录（OHLC + 成交额 + 成交量）
- 已覆盖 521 只 ETF，数据到 2026-06-01

输出: `data/etf_flow.csv`（含资金流向代理指标 `net_flow` / `flow_mult`）

脚本: `scripts/build_etf_flow_from_tdx.py`
Bundle 更新: `scripts/update_bundle_etfs_from_tdx.py`

### 概念 → ETF 映射

Built-in mapping of A-share concept/industry ETFs, covering 30 unique ETFs across 57 concept keywords.
Output: `data/concept_etf_map.csv`

### Block/Industry Stock Lists — 通达信 TDX

Files: `D:\new_tdx\T0002\blocknew\*.blk`
- Plain text, stock codes with market prefix
- `1600186` → `600186.XSHG` (SH), `0002152` → `002152.XSHE` (SZ)
- Covers concept, industry, region, and style blocks

## Online Fallback: AKShare

The project includes `scripts/fetch_data_akshare.py` which can supplement data from:

- ETF snapshots: `fund_etf_spot_em`
- ETF history: `fund_etf_hist_em`
- Concept flow: `stock_fund_flow_concept` / `stock_concept_fund_flow_hist`
- Index proxy for AMV: `stock_zh_index_daily_em` (SSE composite or similar)

Limitations: historical backfill quality varies, Eastmoney endpoints may rate-limit.

## RQAlpha Bundle

Price data for backtesting: `bundle/bundle/` (pre-downloaded RQAlpha public bundle)

## Recommended Workflow

After market close:

```powershell
.\scripts\update_data.ps1                  # All-in-one: AMV + concepts + ETF
.\scripts\run_backtest.ps1
```

Or step by step:

```powershell
# 1. Extract local data from installed software
python scripts/extract_amv_klines.py           # 0AMV from 指南针
python scripts/build_concept_returns.py        # Concept daily returns from TDX
python scripts/build_etf_flow_from_tdx.py      # ETF OHLC + flow from TDX

# 2. Update RQAlpha bundle for recent data
python scripts\update_bundle_etfs_from_tdx.py  # Extends funds.h5 + indexes.h5

# 3. Run backtest and update dashboard
.\scripts\run_backtest.ps1
```

Or with AKShare fallback (if API available):

```powershell
.\scripts\update_data.ps1                  # AKShare supplement + TDX fallback
.\scripts\run_backtest.ps1
```
